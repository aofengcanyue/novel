# 《诡仙录》资源

## 4.1 资源体系三层

第一层：明线通用修仙资源

- 所有修士都能用。
- 用于补灵、养脉、突破、炼器、制符。

第二层：诡仙专属资源

- 温养裂印、施展旧术、布诡局所需。
- 通常来自旧庙、残坛、香火、死局、怨愿、被抹去的遗痕。

第三层：媒介型资源

- 不一定直接提升修为，而是作为布置旧术和做局的载体。
- 例如愿纸、符底、骨钉、灯灰、残坛土。

## 4.2 明线基础资源

- 灵石：通用交易与修炼资源。
- 灵米、兽肉、药汤：维持低阶修士身体和灵力。
- 补气散、养脉丸：补灵、养脉、辅助感气开脉。
- 低阶符材：制符、布阵、辅助斗法。
- 法器残材：修补、炼器、换取资源。
- 妖兽材料：皮、骨、血、筋，可入药、制符、炼器。

## 4.3 诡仙基础资源

香火灰：

- 来源：旧庙、残香、供桌、祭坛。
- 用途：观残、问灰、藏气、辨别香火线残痕。
- 第一卷核心资源之一。

旧愿纸：

- 来源：镇民旧愿、烧残的愿纸、庙中愿册。
- 用途：引错、借怨、显现未尽之愿。
- 适合连接死者情绪与案件线索。

阴骨钉：

- 来源：坟地、旧庙镇物、邪祟封禁。
- 用途：固定小局、压住气机、牵引怨痕。
- 第一卷可先拿到，不必立刻完全会用。

残坛土：

- 来源：废坛、旧祀、倒塌供台。
- 用途：借地势、藏气、布置简陋小局。
- 很适合前期躲查。

灰烬珠：

- 来源：灾后余烬、愿力烧尽后凝成。
- 用途：储存残痕、辅助问灰、承接小型反噬。

## 4.4 中后期诡仙资源

- 血愿玉：由血债与愿力纠缠而成，适合借怨、移祸。
- 裂名符骨：涉及“名”的旧符骨，适合裂名、锁名。
- 旧庙灯芯：承旧香火，可照残痕、引旧愿。
- 众生愿火：大量愿力汇聚后的高阶资源，可撬动香火线。

## 4.5 高阶遗脉资源

- 仙蜕残片：高阶存在遗留，牵涉仙名真相。
- 旧天诏痕：被抹去的古老敕令残痕。
- 无名牌位：没有名字却仍受供的牌位，可牵出失名之人。
- 愿海沉香：沉积大量愿力与旧债的高阶资源。

## 4.6 资源命名与量化

建议采用“品种制 + 品阶制 + 数量单位”。

资源大类：

- 骨类：哭山骨、伏灯骨、碎愿骨、背愿骨、断仙骨、逆名骨。
- 灰类：守夜灰、断明灰、冷香灰、百供灯灰、反愿香灰、镇名灰。
- 血类：葬灯血、凶祟血、压煞血、噬愿血、逆命血、夺名血。
- 符类：藏痕符底、裂纹残符、逆纹符皮、封名符诏。
- 愿类：残愿丝、灯愿丝、断供丝、反愿丝、失名丝、立誓丝。

数量单位：

- 骨类：枚、截、根。
- 灰类：撮、盏、匣。
- 血类：滴、盅、瓶。
- 符类：张、页、卷。
- 愿类：缕、束、团。

第一卷核心资源：

- 香火灰。
- 残坛土。
- 旧愿纸。
- 阴骨钉。

## 4.7 资源量化执行口径

资源体系采用四层表达：

1. 大类：骨、灰、血、符、愿。
2. 品种名：正文真正出现的名字，如守夜灰、伏灯骨、藏痕符底。
3. 品阶：凡品、灵品、真品、道品。
4. 数量单位：一捧、一匣、一截、一页、一缕等。

品阶口径：

- 凡品：常见低阶货，能救急、做小局、撑前期。
- 灵品：低阶修士真正会争的资源，能稳定术法或辅助突破。
- 真品：中期关键资源，能支撑完整布局或裂印阶段跃迁。
- 道品：后期资源，开始触碰“名”“法统”“仙名锚点”。

大类用途标签：

- 骨类：偏承载。用于做骨器、承诡气、固定局眼、补裂印。
- 灰类：偏遮掩。用于藏气、遮痕、稳印、布置旧术。
- 血类：偏催发。用于爆发、催术、移祸、短时改途，反噬也重。
- 符类：偏布置。用于刻诡纹、伪装现场、锁局、改残迹。
- 愿类：偏观与名。用于观痕、触香火、看供奉、动仙名。

正文表达尽量使用“品种 + 品阶 + 单位”，例如：

- 一匣灵品守夜灰。
- 半截真品背愿骨。
- 一页真品逆纹符皮。
- 一缕灵品残愿丝。

这样读者能同时知道：它是什么、值不值钱、够不够用。

## 4.9 资源与裂印阶段绑定

得印阶段：

- 常用资源：守夜灰、伏灯骨、藏痕符底、残愿丝、葬灯血。
- 资源特点：便宜、常见、够活命。
- 剧情用途：躲查、藏气、第一次观痕、第一次引错。

识印阶段：

- 常用资源：哭山骨、断明灰、逆纹符皮、逆命血、断供丝、背愿骨。
- 资源特点：开始变贵，能支撑完整小局。
- 剧情用途：借势、移祸、改现场、让仇人自己入局。

掌印 / 裂名阶段：

- 常用资源：逆名骨、镇名灰、夺名血、封名符诏、失名丝、旧天诏痕。
- 资源特点：不只是修术，而是开始动“名”和“法统”。
- 剧情用途：锁名、裂名、审仙名、撬动香火锚点。

交易语言示例：

- “一滴真品逆命血，换三十块中品灵石。”
- “这块伏灯骨最多灵品，值不了两匣守夜灰。”
- “背愿骨不是你这境界能碰的，拿不好会反噬。”
- “他这趟黑市最大的收获，不是灵石，是那页逆纹符皮。”

资源写法原则：

- 资源必须推动剧情，不能只当战利品堆仓库。
- 每次获得资源，最好对应一次新用法、新风险或新线索。
- 主角前期应常常“不够用”，这样每一捧灰、一页符底才有重量。
- 稀有资源出场要连着人物、地点或旧案，不要凭空掉落。

## 4.10 俗称、真名与古称

资源命名采用双层到三层结构：

1. 俗称：黑市、散修、低阶修士常用，读者容易懂。
2. 真名：古籍、旧脉、强者口中使用，带世界观气质。
3. 古称 / 禁称：高阶资源或涉及仙名真相时使用，前期只露碎片。

示例：

| 资源本体 | 俗称 | 真名 | 古称 / 禁称 |
| --- | --- | --- | --- |
| 骨类 | 诡骨 / 阴骨 | 残仙骨 / 断劫骨 | 断代仙骸 / 命碑骨 |
| 血类 | 煞血 / 祟血 | 逆命血 / 殉劫真血 | 命火真血 / 不朽灾血 |
| 灰砂类 | 灯灰 / 阴砂 | 仙陨灰 / 命香灰 | 灭灯劫砂 / 古庙天灰 |
| 符皮类 | 旧符皮 | 蜕符皮 / 命契古皮 | 封劫符卷 / 断代法诏 |
| 灯香类 | 残香 / 灯芯 | 祭仙古香 / 命灯芯 | 诡仙香火 / 葬仙灯种 |
| 露髓类 | 阴露 / 骨髓液 | 劫髓 / 断命髓 | 仙蜕真露 / 诡仙道髓 |

使用节奏：

- 前期多用俗称和具体物件，如香火灰、旧愿纸、残坛土、阴骨钉。
- 中期通过黑市、旧庙、残经或高阶人物揭出真名。
- 后期再使用古称 / 禁称，绑定旧脉真相、仙名锚点和大机缘。

成长感示例：

- 前期：“这是一截诡骨。”
- 中期：“不是诡骨，是残仙骨。”
- 后期：“残仙骨只是后人俗称，它原本叫断代仙骸。”

## 4.11 暗线高阶资源树

主角暗线体系的高级资源按用途分四类。

养印资源：

- 命香灰。
- 仙陨灰。
- 葬印灰。
- 残坛秘砂。
- 诡仙香火。
- 劫庙灯芯。

用途：

- 温养裂印。
- 稳定观残、藏气、引错的反噬。
- 解锁更深旧术。

炼体资源：

- 残仙骨。
- 断劫骨。
- 逆命血。
- 断命髓。
- 葬脉灵露。
- 祟仙遗血。

用途：

- 强化肉身承载力。
- 抵御裂印反噬。
- 补足明线修为不足带来的身体短板。

施术资源：

- 蚀纹砂。
- 蜕符皮。
- 封劫符卷。
- 守灵灰。
- 引祟香。
- 照骨灯油。

用途：

- 画符、布局、遮痕、寄祟。
- 给“观、藏、借、改”提供具体载体。

破境资源：

- 劫髓。
- 仙蜕真露。
- 命火真血。
- 诡仙道髓。
- 葬仙灯种。
- 不朽灾血。

用途：

- 冲击裂印阶段或明线大境界。
- 完成得印到识印、掌印到裂名等关键跃迁。
- 每次使用必须绑定重代价，不能当普通经验丹。

写法提醒：

- 高阶资源前期只适合以残字、传闻、旧册名目出现。
- 真正入手高阶资源时，要连着旧案、禁地、人物债或香火黑线。
- 不要让资源名风格乱跳，统一保持“破败上古、祭祀余烬、命数诡变”的气质。

## 4.12 交易与违禁口径

资源流通要服务中期剧情，尤其是县城、庙观、黑市和巡香司外围身份。

明面可交易：

- 灵石。
- 灵米、兽肉、药汤。
- 补气散、养脉丸。
- 低阶符材。
- 法器残材。
- 普通妖兽材料。

灰色可交易：

- 少量香火灰。
- 旧愿纸边角。
- 废庙灯灰。
- 低阶骨钉。
- 供册边角料。
- 被废弃的符底。

这些东西明面上不一定违法，但来源难解释，容易牵出庙观、义庄、巡香司和黑市。

违禁资源：

- 正式供册缺页。
- 香火账簿碎页。
- 勘祠案词残页。
- 仙陨灰、命香灰等涉及旧庙或仙祠核心的资源。
- 裂名符骨、旧天诏痕、无名牌位等涉及名与法统的资源。

持有违禁资源的后果：

- 被巡香司盯上。
- 被定为盗祠、邪修或魔修余脉。
- 被宗门借口带走审查。
- 被黑市中人敲诈或出卖。

交易写法原则：

- 价格不是重点，风险才是重点。
- 每次交易最好带出一个新流程：谁能卖、谁敢买、谁负责验货、谁会告密。
- 主角买资源时，同时也在买线索、身份和进入下一层局的门票。
