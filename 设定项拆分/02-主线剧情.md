# 《诡仙录》主线剧情

## 2.1 主线链条

边荒小镇“清平镇”供奉仙门  
→ 主角送供归来，发现镇子化为废墟  
→ 正道修士争功，无人救人  
→ 被追杀的魔修临死压偏杀势，主角从死人堆里活下来  
→ 主角为全镇善后，收尸、掩埋、清理仙祠，扶正供台时触发神像裂缝里的“诡仙裂印”残痕  
→ 裂印认主，主角被卷入香火线追查  
→ 主角靠观痕、藏气、引错、借势求活  
→ 从私仇追查到地方庙观、宗门功绩、香火秩序  
→ 发现诡仙遗脉被抹去的真相  
→ 从复仇者成长为重定“仙”之名的人。

## 2.2 开篇核心事件

开篇方向：灾后开篇，灰烬中醒来。

开篇不再按“清晨开祠 → 送供上山 → 归来见废墟”的正序慢铺，而是直接从惨案之后切入：主角从死人堆里醒来，仙祠半塌，神像开裂，供灯未灭，熟悉的人都已经死去。

前三章必须完成三个钩子：

1. 主角怎么活下来的。
2. 这场惨案是谁造成的。
3. 诡仙裂印为什么会落到“埋人的人”手里。

开篇结构：

1. 灰烬中醒来：主角从废墟死人堆里活过来，先给惨案画面和生存钩子。
2. 仙人在上：以半昏迷回忆或残痕闪回补出正道争功、魔修压偏杀势、凡人无人救的核心反差。
3. 埋人的人得到：主角为全镇善后，扶正供台时触发神像裂缝里的裂印残痕，完成金手指出场。
4. 第一次观痕：裂印让主角看见被抹掉的真相残迹。
5. 有人回来清理：香火线查痕人出现，确认这场灾不是结束。

原本的“清晨开祠、镇民许愿、老庙祝说话、送供上山”不删除，改成前三章里的碎片回忆：

- 主角看见裂开的神像，想起早上自己刚擦过神像。
- 主角摸到烧残愿纸，想起阿禾写过愿。
- 主角看见供灯未灭，想起老庙祝说过“人敬仙，不过是盼着有难时，他们肯看顾这一眼。”
- 主角听见“凡民死伤，不足记”，反打凡人敬仙的朴素信念。

开篇画面魂：

> 主角替全镇去敬仙，回来时却看到仙人留下的死地。

情绪钉子：

> 香火还没散，拜仙的人已经死光了。

### 2.2.1 惨案现场关键画面

归来见废墟要一层层推进，不要一眼把全部信息倒完：

1. 静：山路太安静，鸟雀不鸣，风里有焦味。
2. 痕：树木有术火烧痕，迎仙幡倒在泥里，符纸碎裂。
3. 祠毁：石阶断裂，祠门塌半扇，神像从中裂开，香炉碎裂，香灰混着血。
4. 认人：先认出一两个熟人，再让惨案扩大成整镇死寂。
5. 供奉与死亡重叠：供果滚进血泊，长明灯照着碎碗和焦黑的人手。

主角最好看到大战尾声，而不是只看结果：

- 正道修士还在争谁斩魔、谁记功、谁得遗物。
- 有人说“先定此獠归属，莫叫旁宗抢了头功”。
- 或者更狠：“死些凡人算什么，若让他走脱，才真是大过。”

一句话足够打碎主角心里的仙。

### 2.2.2 被追杀者的规矩

被追杀者不能写成圣人。

他的定位是：不是好人，但在这一刻比那些“仙人”更像人。

落地方式：

- 他已经重伤到极点，知道自己走不掉。
- 他看见四周是凡俗聚落，正道修士仍在封路争功。
- 他不是想救苍生，而是不愿借凡人骨血给这群人铺功名。
- 他在最后绝手爆开前，强行压偏最致命的一道杀势，给凡地留下一角生机。

可用台词方向：

> 你们口口声声除魔，原来除的是挡你们升阶的魔。

## 2.3 主角定位

暂定身份：边荒小镇仙祠香童 / 庙祝学徒。

职责：

- 添灯。
- 抄愿。
- 守供。
- 协助祭祀。
- 送供跑腿。

前期气质：

- 小人物。
- 见过人间苦。
- 能忍、能看、能撑。
- 不热血莽撞，也不彻底黑化。
- 先求活，再复仇，再追问，最后立道。

主角名备选：沈逐、林渡、顾停灯。当前工作名可先用“沈逐”。

## 2.4 主角幸存标准口径

主角活下来，不等于被拯救。

推荐组合版：

1. 主角去送供，不在最初杀劫中心。
2. 他回程赶上大战尾声，被术法余波卷入。
3. 被追杀者临死压偏杀势，护下一角生机。
4. 主角仍然被震伤脏腑、神魂受创，埋进废墟与尸堆之间。
5. 他过了一夜或数日才从死人和香灰里爬出来。

这个口径的意义：

- 他不是幸运儿，而是从旧世界的尸体里醒来。
- 他不是被仙救的，甚至不是被魔修主动救的。
- 他活下来，是因为所有人都不把凡人当回事时，唯一有规矩的人留了一寸生路。
- 这会让他对魔修有复杂情绪，却不至于直接站到魔道一边。

幸存者数量口径：

- 不是全镇只有主角一个人活着。
- 但仙祠核心废墟区，主角是唯一从核心杀势里爬出来的人。
- 镇外、义庄、山路、猎户屋可留下少量边缘幸存者，用来提供证词和线索。
- 前期幸存者不宜超过三到五个，避免削弱惨案冲击。

推荐幸存者：

- 陈瘸子：当时在镇外义庄，能看尸体异常；第一卷中段会因白事簿和替主角揽痕被灭口，死后以白事簿残页继续留证。
- 韩猎户：当时在山里，能提供旧庙与山路目击。
- 周哑女：被藏在地窖或柴房，能提供非语言线索。
- 一名临死镇民：只短暂存活，留下“不是妖祟”的遗言。

## 2.5 机缘获取标准口径

机缘不能写成“魔教大佬临死传功”。

最合适的逻辑：

- 机缘不是传功，而是残留的路。
- 主角不是抢到，也不是被送到手里，而是在善后行为中触发。
- 修士争夺机缘找不到，埋人的人反而得到。

推荐形式：

- 裂印：裴烬带走或触动的诡仙旧印残痕被大战打入神像裂缝，主角善后扶供台时以血、灰、旧愿纸触发，后来寄入主角命数。
- 半卷残经：被血浸透、被火烧去一半，只剩总纲和几句离经叛道的话。
- 命灯残火：仙祠供灯沾上异道气机，成为诡火意象。可作为辅助意象，不必抢裂印核心。

最稳组合：裂印 + 半卷残经。

主角得到的不是奖励，而是一个问题：

> 仙若无情，魔若有规，那什么才是正？

## 2.6 主角成长线

第一阶段：复仇

- 目标：找出直接造成小镇惨案的人。
- 情绪：恨、痛、活下去。
- 行动：躲查、找线索、第一次做局。

第二阶段：追问

- 目标：弄清惨案为什么会发生。
- 情绪：从私仇进入认知怀疑。
- 行动：追查香火线、庙观线、宗门功绩线。

第三阶段：否定

- 目标：确认“正道是否真的正”。
- 情绪：信仰崩塌，开始质疑仙名秩序。
- 行动：揭开更多被掩盖的凡人惨案和诡仙旧事。

第四阶段：立道

- 目标：重定“仙”之名。
- 情绪：从复仇者变成执道者。
- 行动：以诡仙路撬动高层秩序，让仙名重新面对人间。
