# 《诡仙录》长篇推进指导

## 12.1 总体判断

当前设定可以支撑 150—200 万字，若要稳定支撑 200 万字以上，关键不是继续堆世界观，而是把已有设定拆成可持续连载的剧情发动机。

现有强项：

- 世界核心矛盾清楚：仙凡供奉、香火秩序、功绩体系。
- 主角成长线清楚：复仇 → 追问 → 否定旧秩序 → 重定仙名。
- 力量体系清楚：明线修为 + 暗线裂印。
- 修炼路径清楚：废墟自修 → 伪装入城 → 借名入宗 → 脱宗入旧脉 → 自立道统。
- 人物层次清楚：清平镇幸存者、巡香司、正道裂缝人物、逆灯门。
- 分卷结构清楚：六卷长线。

当前需要补强的不是“设定概念”，而是：

- 每卷内部阶段目标。
- 地图和势力层级。
- 反派梯队。
- 资源与剧情绑定。
- 主角每 20—30 万字一次的能力质变。

## 12.2 目标体量

按 200 万字规划：

| 卷数 | 卷名 | 建议字数 | 核心功能 |
| --- | --- | --- | --- |
| 第一卷 | 仙祸临门 | 20—25万 | 开篇钩子、得印、求活、旧庙局杀 |
| 第二卷 | 诡路寻仇 | 25—30万 | 入县城、第一层复仇、正道人物入场 |
| 第三卷 | 香火成网 | 35—40万 | 展开供奉链、功绩簿、地方庙观网络 |
| 第四卷 | 旧脉重现 | 35—40万 | 诡仙旧脉、残碑断坛、掌印到裂名 |
| 第五卷 | 问仙于世 | 35—40万 | 公开撕开仙名遮羞布，正道内部分裂 |
| 第六卷 | 重定仙名 | 35—45万 | 终局、执劫、重定仙名 |

总体章节参考：

- 若每章约 2500 字：约 800 章。
- 若每章约 3000 字：约 650—700 章。
- 若每章约 4000 字：约 500 章。

建议网文节奏：

- 每 3—5 章要有一个小钩子。
- 每 10—15 章要有一次小推进或小爽点。
- 每 25—35 章要完成一个阶段目标。
- 每卷末必须有一次局势升级，而不是只打赢一个敌人。

## 12.3 长篇推进总原则

每卷必须同时推进六条线：

1. 主线真相：清平镇惨案背后的香火、功绩、仙名秩序。
2. 主角修为：明线境界逐步补足，不能只靠裂印。
3. 裂印层级：从得印到识印、掌印、裂名、照骨、执劫。
4. 人物关系：顾清棠、许观澜、陆承言、裴守拙等正道裂缝人物持续变化。
5. 地图层级：清平镇 → 县城 → 府城 → 宗门外围 → 旧脉禁地 → 仙名核心。
6. 反派梯队：清痕人 → 香火使 → 庙观主事 → 功簿司 → 宗门高层 → 仙名秩序。

每一阶段不能只换敌人，要换三样东西：

- 新身份。
- 新资源。
- 新规则。

长篇反派硬原则：

- 不能靠降智衬托主角。敌人越聪明、越专业、越会做局，主角赢下来的半步才越有含金量。
- 反派强不只体现在修为和法宝上，还要体现在判断、组织、预案、复盘、心理压迫和对规则的利用上。
- 每个重要反派至少要有一个“正确判断”：他可以不知道全部真相，但不能对眼前证据视而不见。
- 主角的小胜应来自信息差、代价、环境、敌人经验惯性和制度缝隙，不来自敌人突然犯蠢。
- 反派每吃一次亏，都要升级一次手段：补流程、换人查、设诱饵、反向钓鱼、转移证据、灭口、请上级。
- 长篇推进的精彩感来自“主角和敌人都在变强”，不是主角单方面碾压越来越笨的对手。

例如：

- 第一卷：幸存者身份，香火灰资源，清痕规则。
- 第二卷：假身份入城，供册 / 低阶符材资源，县城庙观流程规则。
- 第三卷：协查 / 外围客卿身份，功簿 / 巡境图录资源，宗门记功规则。
- 第四卷：旧脉继承者身份，旧庙灯芯 / 残碑资源，诡仙遗脉规则。
- 第五卷：问仙者身份，众生愿火资源，仙名公开冲突规则。
- 第六卷：执劫者身份，仙名锚点资源，重定规则。

## 12.4 卷内阶段拆分模板

每卷建议拆成 4—5 个小篇章。

每个小篇章都要回答：

- 当前主角要什么？
- 当前敌人要什么？
- 当前资源是什么？
- 当前信息差是什么？
- 本段爽点是什么？
- 本段结尾抬出什么更大问题？

标准小篇章结构：

1. 入局：主角进入新地点 / 新身份 / 新任务。
2. 探查：发现流程、规则、敌人、资源。
3. 受压：敌人逼近，主角旧办法不够用。
4. 学会：主角掌握一个新技巧 / 新资源用法。
5. 做局：主角用新理解反打。
6. 抬格局：胜利只解决表层，露出更深一层。

## 12.5 六卷阶段目标

### 第一卷：仙祸临门

阶段目标：

1. 前 10 章：灾后醒来、正魔反差、善后得印、第一次藏气、旧庙线登场。
2. 11—20 章：孟香使压力升级、第一次引错、裂印反噬、阿禾和旧庙线推进。
3. 21—35 章：反查孟香使路线，收集第一批诡仙资源。
4. 36—50 章：旧庙局杀，卷尾露出祝司首 / 地方香火线。

能力质变：

- 明线：普通人 → 感气 / 开脉边缘。
- 暗线：得印稳定。
- 旧术：观残、藏气、引错、借势雏形。

卷末必须完成：

- 主角从幸存者变成入局者。
- 确认惨案不是孤立事件。
- 第一次完整诡道局杀成立。

### 第二卷：诡路寻仇

阶段目标：

1. 入城藏身：主角用假身份进入县城，摸庙观外围。
2. 正道入场：顾清棠、许观澜正式出现，第一次对峙。
3. 案卷错位：死者名单、供奉账册、宗门记录开始对不上。
4. 陆承言压事：体制型妥协者出现，把不对的事说得合理。
5. 做局复仇：主角扳倒或杀掉第一层仇人。

能力质变：

- 明线：感气稳定 → 开脉 / 筑基门槛。
- 暗线：得印 → 识印。
- 旧术：引错稳定，借势成型。

卷末必须完成：

- 第一层复仇。
- 顾清棠开始不信宗门记录。
- 许观澜第一次被现实刺裂。
- 陆承言“温和共犯”形象成立。
- 功簿 / 供册线索指向第三卷。

### 第三卷：香火成网

阶段目标：

1. 府城 / 大庙观登场：香火网络扩大。
2. 功簿司露面：除魔记功体系具体化。
3. 供奉链查清：凡人如何被香火、庙观、宗门共同裹挟。
4. 正道内部分裂：顾清棠、许观澜、陆承言立场持续拉开。
5. 中型布局：主角让一处庙观体系短暂失灵。

能力质变：

- 明线：筑基 → 灵府门槛。
- 暗线：识印 → 掌印门槛。
- 旧术：移祸、归祸雏形，开始能对流程做局。

卷末必须完成：

- 主角从复仇者转向追问者。
- 香火体系不是个别恶人，而是一张网。
- 诡仙旧脉线索指向旧庙、残碑、断坛。

### 第四卷：旧脉重现

阶段目标：

1. 旧庙遗址群登场。
2. 残碑、断坛、无名牌位逐步揭示诡仙遗脉。
3. 主角确认诡仙不是魔道分支，而是被抹去的另一条仙路。
4. 逆灯门余部追索裂印，正魔两边都不可信。
5. 高位存在开始察觉裂印。

能力质变：

- 明线：灵府 → 金丹门槛。
- 暗线：掌印稳定，触碰裂名。
- 旧术：藏因、锁痕、借名、归祸稳定。

卷末必须完成：

- 诡仙遗脉真相初步成立。
- 主角从借壳者变成旧脉继承者。
- “谁有资格称仙”的问题浮出水面。

### 第五卷：问仙于世

阶段目标：

1. 仙城 / 法会 / 大祭场景登场。
2. 主角不再只暗中杀人，而是让事实显形。
3. 香火线与功绩线合流。
4. 正道内部公开分裂。
5. 主角让一地人不再盲信某个仙名。

能力质变：

- 明线：金丹 → 元婴 / 化神门槛。
- 暗线：裂名 → 照骨。
- 旧术：审仙名、扰真名、显旧罪。

卷末必须完成：

- 主角成为旧秩序公开通缉的异数。
- 高层仙名秩序第一次正面回应。
- 顾清棠等正道人物必须做出公开选择。

### 第六卷：重定仙名

阶段目标：

1. 仙名秩序核心场景登场。
2. 众生供奉、愿力、真名、功绩、正统之间的最终结构浮出水面。
3. 主角承接大因果，进入执劫者状态。
4. 终局不是战力对轰，而是因果、香火、仙名和众生愿力的总局。
5. 重定“仙若受香火，该承担什么”。

能力质变：

- 明线：化神以上，具备硬实力。
- 暗线：照骨 → 倒果 / 执劫。
- 旧术：断正统、改归属、重定名。

卷末必须完成：

- 诡仙旧脉不再只是禁忌残脉。
- 主角不是登顶强者，而是新路开道者。
- 老庙祝的朴素话语完成终局回收。

## 12.6 地图层级规划

地图扩展应服务主线，不要无意义换地图。

建议层级：

1. 清平镇：开篇惨案、得印、求活。
2. 县城：伪装入城、第一层复仇、正道人物入场。
3. 府城：香火网络、庙观联盟、功簿司外层。
4. 玄霄宗外围：外门、巡律堂、功簿司、试炼场。
5. 大庙观 / 香火总司：供奉链核心节点。
6. 旧庙遗址群：诡仙遗脉真相。
7. 封禁旧脉禁地：残碑、断坛、无名牌位。
8. 仙城 / 法会：公开问仙，仙名与事实冲突。
9. 仙名秩序核心：终局。

每个地图要绑定：

- 一个新规则。
- 一种新资源。
- 一个新反派。
- 一次主角身份变化。

## 12.7 反派梯队规划

反派不能只有“坏人”，要有层级和功能。

| 层级 | 代表 | 功能 |
| --- | --- | --- |
| 清痕执行层 | 罗灰 | 让主角第一次用引错 |
| 地方收尾层 | 孟香使 | 第一卷直接生存压力 |
| 地方香火层 | 祝司首 | 第一层黑线，连接县城 |
| 正道争功层 | 许照骨 | 证明惨案来自功绩体系 |
| 温和共犯层 | 陆承言 | 证明恶不只来自恶人，也来自沉默和压事 |
| 功簿记录层 | 商不言 / 功簿司人物 | 把死人写成合法数字 |
| 香火网络层 | 府城香火总执 | 展开香火成网 |
| 旧脉误用层 | 逆灯门余部 / 夺印者 | 证明魔道也不可靠 |
| 宗门高层 | 玄霄宗掌权者 | 把功绩、香火、资源捆成秩序 |
| 仙名秩序层 | 高位仙名存在 | 终局对手 |

每个反派都要回答：

- 他想保什么？
- 他怕什么暴露？
- 他如何代表一层规则？
- 主角击败他后，世界会露出哪一层更深问题？

## 12.8 资源推进规划

资源不能只做仓库清单，要做剧情发动机。

第一卷：

- 香火灰、旧愿纸、残坛土、阴骨钉。
- 用途：观残、藏气、引错、小局。

第二卷：

- 供册残页、低阶符材、养脉丸、法器残材。
- 用途：明线修为补足，县城伪装和查账。

第三卷：

- 功绩簿、巡境图录、庙观香火节点。
- 用途：揭开供奉链和记功体系。

第四卷：

- 旧庙灯芯、裂名符骨、无名牌位、诡仙残碑。
- 用途：进入旧脉，掌印到裂名。

第五卷：

- 众生愿火、旧天诏痕、镇名灰。
- 用途：公开问仙，让仙名与事实冲突。

第六卷：

- 仙名锚点、愿海沉香、执劫资格。
- 用途：终局重定仙名。

## 12.9 能力质变规划

长篇必须定期给主角一次能力质变。

建议节奏：

第一卷末：

- 得印稳定。
- 能完成小型引错和旧庙局杀。

第二卷末：

- 识印开启。
- 主角能主动借势，不再只被动逃命。

第三卷末：

- 掌印门槛。
- 主角能对一处流程 / 小体系做局。

第四卷末：

- 掌印稳定，裂名雏形。
- 主角开始触碰“名”的层面。

第五卷末：

- 裂名 / 照骨。
- 主角能公开撕开仙名遮羞布。

第六卷：

- 倒果 / 执劫。
- 主角能承接大因果，重定规则。

每次质变都必须付代价：

- 因果债。
- 心神磨损。
- 暴露风险。
- 真我偏移。
- 身边人受牵连。

## 12.10 第三卷是长篇成败关键

第一卷和第二卷天然好写：

- 第一卷有惨案、求活、得印。
- 第二卷有入城、复仇、女主登场。

真正决定 200 万字能不能稳住的是第三卷《香火成网》。

第三卷必须写活：

- 供奉链。
- 功绩簿。
- 庙观网络。
- 正道内部分裂。
- 主角借壳入宗。
- 顾清棠查案。
- 陆承言压事。
- 商不言改账。
- 许观澜信念裂开。

如果第三卷立住，后面第四卷旧脉、第五卷问仙、第六卷重定仙名就都有根。

第三卷不能只写“查大阴谋”，要写具体流程：

- 谁收供？
- 谁登记？
- 谁改账？
- 谁记功？
- 谁拿资源？
- 谁压死者名单？
- 谁用“正道大局”说服自己沉默？

## 12.12 最终指导口径

《诡仙录》的长篇推进，不靠无限换地图打怪，而靠三件事不断升级：

1. 主角看见更深一层真相。
2. 主角借用更复杂一层规则。
3. 主角付出更重一层代价。

每一卷都要让读者感觉：

> 原来清平镇不是例外，只是这套仙道秩序露出的第一道裂缝。
