# 《诡仙录》势力设定

## 10.1 地点口径：清平镇

开篇地点建议定为“镇”，不是普通村庄。

工作名：清平镇。

理由：

- 镇比村更适合承载仙祠、岁供、庙祝、义庄、猎户、纸扎铺、外来商旅、巡香司等要素。
- 镇有周边村落依附，可以自然形成“镇中核心区 + 镇外边缘幸存者 + 山路旧庙”的空间层次。
- 镇能解释为什么会有正式供奉名册、巡境灵牌、岁供队伍和香火账目。

清平镇结构：

- 镇心：仙祠、供台、长明灯、愿册、岁供存放处。惨案核心区。
- 镇街：米铺、药铺、纸扎铺、猎户常出入的酒摊等。多数被毁。
- 镇外：义庄、坟地、柴道、猎户山屋。可留下少量边缘幸存者。
- 北山旧庙：疑似诡仙遗脉旧痕所在，也是第一卷旧庙局杀的核心场景。
- 迎仙台 / 山门路：送供队伍上山交供的地方，保持仙门距离感。

## 10.2 幸存者口径

不要写成“全镇只有主角一个人活着”，那样线索、人间余味和后续人物会太少。

建议口径：

- 仙祠核心废墟区：主角是唯一从核心杀势里爬出来的人。
- 镇中主街：几乎全灭，只有极少数重伤者短暂存活，方便留下遗言或遗物。
- 镇外边缘：有少量幸存者，因为他们当时不在镇心或被地形隔开。
- 外出未归者：可保留一两人，如韩猎户，制造目击线索与嫌疑。

前期幸存者控制在三到五个以内：

- 沈逐：核心幸存者，善后得裂印。
- 陈瘸子：镇外义庄 / 白事铺幸存，懂尸体和白事。
- 韩猎户：外出未归的可疑目击者。
- 周哑女：被藏在地窖或柴房的幸存小姑娘，可提供非语言线索。
- 一名临死镇民：只活半章，用来留下关键遗言或指认“不是妖祟”。

这样惨案仍然足够狠，但故事不会失去证词、线索和人间回声。

## 10.3 正道宗门：玄霄宗

玄霄宗是清平镇名义上的庇护宗门。

宗门定位：

- 地方正道大宗。
- 接受清平镇岁供、供灯、香火与谢愿牌。
- 通过巡境灵牌和护境名册确认治下。
- 对外宣称护持一方、除魔卫道。

内部问题：

- 除魔记功体系成熟。
- 功绩可以换资源、身份、晋升名额。
- 低阶弟子和外门执事容易为了抢功忽视凡人。
- 高层未必人人恶，但整个制度默认“凡民损耗可被记账”。

相关部门：

护境堂：

- 名义职责：巡境、护民、处理妖祟。
- 实际问题：只重“是否完成任务”，不重凡人伤亡。

斩魔司：

- 名义职责：追杀邪修、夺回禁物。
- 开篇追杀裴烬的主力。
- 与功簿司关系紧密，最容易争功。

功簿司：

- 记录斩魔功绩、资源分配和晋升凭证。
- 可成为第三卷“香火成网”的关键机构。

巡律堂：

- 负责调查宗门内部失职、错案和异常记录。
- 顾清棠可出自此处，代表正道内部尚未死透的理想。

## 10.4 地方香火线：巡香司

巡香司是连接王朝地方行政、庙观香火体系与玄霄宗治下供奉秩序的灰色机构。

名义权属：

- 对王朝来说，它是处理仙祠、祭祀、香火异案的特殊事务接口。
- 对庙观来说，它是维护供册、愿册、祭典记录和地方香火秩序的办事层。
- 对玄霄宗来说，它是治下香火脉和庇护叙事的外围执行手。

因此巡香司不宜写成纯宗门直属衙门，也不宜写成普通县衙差房。它最有味道的地方，是同时有官面合法性、庙观熟门路和宗门背书。

表面职责：

- 管理仙祠、供灯、岁供、愿册。
- 核验各地香火是否稳定。
- 维护巡境灵牌与护境名册。
- 勘验仙祠崩毁、祭祀异常、香火断绝等事件。
- 向县衙、庙观和宗门递交不同口径的文书。

暗中职责：

- 清理不利于宗门正统叙事的现场。
- 改写凡人死伤记录。
- 回收愿纸、香火灰、供册、残符。
- 将复杂事件统一归因为“魔修作乱”。

第一卷代表人物：

- 孟香使：清痕执行者，专业收尾人。
- 罗灰：孟香使手下，专查香火灰、愿纸、供台残痕。
- 祝司首：上层影子人物，第一卷只露命令和痕迹。

巡香司和玄霄宗关系：

- 名义上受玄霄宗管辖。
- 实际上在地方具有很强独立性。
- 很多脏活不由宗门正面弟子做，而由巡香司收尾。
- 王朝基层差役遇到香火异案时，也常被巡香司以“勘祠”“核供册”名义压到外围。

清平镇惨案中的到场逻辑：

- 里正、巡检司差役或报灾小吏可以先到外围，负责封镇、认尸、报灾。
- 他们看见仙祠塌毁和修士残痕后，不敢擅自深入核心。
- 罗灰这类查痕人随后进入核心，回收愿纸、庙灰、供册和供台残物。
- 孟香使再正式接手，把事件从“现场清理”推进到“案词定性”。
- 祝司首只露命令、批痕、铜牌或卷宗影子，第一卷不宜过早正面下场。

详见 [15-王朝与地方权力结构.md](15-王朝与地方权力结构.md)。

## 10.5 魔教 / 异道：逆灯门

正道称其为“逆灯魔门”，残存者内部可能自称“残灯道”或“守灯一脉”。

势力定位：

- 非正道，确实危险。
- 有血法、禁术、夺命手段，不能洗白成善良组织。
- 但一部分核心规矩很硬：不借凡人命，不踏凡人骨，不以香火豢养众生。

与诡仙遗脉关系：

- 逆灯门不等于诡仙遗脉。
- 它可能曾经接触、保存或误用过部分诡仙旧物。
- 裴烬携带的裂印和残经，不一定是逆灯门完整传承，而是他们从旧庙 / 禁地 / 香火司争夺中拿到的残物。

开篇任务：

- 裴烬带着裂印、半卷残经或一页香火功簿残证逃亡。
- 玄霄宗斩魔司追杀他，表面说除魔，实际也要夺回不该外流的东西。
- 裴烬不是为了救清平镇而来，但他临死不愿让凡人骨血给正道铺功名。

后续作用：

- 逆灯门余部可能也会找裂印。
- 他们不一定是主角盟友，甚至会想夺回裂印。
- 他们能让主角看到“魔名之下也有不同规矩”，但不能让主角简单站队魔教。

## 10.6 开篇两方任务

正道方明面任务：

- 追杀逆灯门修士裴烬。
- 夺回禁物。
- 记除魔之功。
- 对外宣布清平镇惨案为魔修作乱。

正道方暗线任务：

- 找回裂印 / 残经 / 香火功簿残页。
- 抹掉正道争功导致凡人死伤的痕迹。
- 修改供奉账册和死伤记录。
- 找出是否有幸存者看见大战尾声。

魔教 / 异道方任务：

- 裴烬要把裂印或残证带出玄霄宗势力范围。
- 他不负责救凡人，也不准备做英雄。
- 但他守住“不借凡人命”的底线，临死压偏杀势。

巡香司任务：

- 清场。
- 回收愿纸、庙灰、供册、残符。
- 找到动过供台或接触裂印的人。
- 将清平镇从“庇护失职”改写成“魔祸损耗”。

主角任务：

- 前十章只做三件事：活下来、弄清谁回来清痕、第一次让追查者走错。

## 10.7 开篇势力人物表

玄霄宗正道方：

- 许照骨：斩魔司执事，开篇追杀裴烬的主要正道修士之一，冷硬功利。
- 许观澜：年轻正道弟子，真信除魔卫道，可能看见不对但还不敢承认。
- 顾清棠：巡律堂弟子，女主工作名，前十章不正式登场，后续奉命复查。
- 陆承言：中层执事，擅长接手案子、统合消息、稳定局势，第二卷中段正式发力。
- 裴守拙：老派守底线者，可先通过批语、传闻、拒签记录埋影。
- 叶知微：顾清棠同门，偏记录、辨伤、查细节。
- 商不言：功簿 / 内务人物，负责把“规则吃人”具体化。

巡香司香火线：

- 孟香使：第一卷直接反派。
- 罗灰：孟香使手下，查灰、查愿纸、查供台残痕。
- 祝司首：孟香使上级，掌管一片地方香火账与清痕命令。

逆灯门 / 残灯道：

- 裴烬：被追杀者，死亡后留下裂印线索。
- 桑婆：逆灯门余部，可中后期登场，知道裴烬真正带走了什么。
- 乌行客：逆灯门追索者，可作为第二卷压力源，不是主角盟友。

清平镇地方人物：

- 沈逐：主角，仙祠香童。
- 老庙祝：死于惨案，留下母题句。
- 阿禾：死于惨案，留下旧愿纸。
- 陈瘸子：镇外幸存，懂尸体和白事。
- 韩猎户：外出幸存，提供旧庙和山路线索。
- 周哑女：边缘幸存，能指出某个清痕人的动作或特征。
