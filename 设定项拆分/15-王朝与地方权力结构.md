# 《诡仙录》王朝与地方权力结构

## 15.1 核心口径

清平镇不是无主荒地，也不是只有宗门在管。

这个世界的地方秩序应按三句话理解：

> 王朝管地、人户、税粮与明面治安。
> 庙观和巡香司管仙祠、香火、祭祀异案与卷宗定性。
> 宗门管修士事务、法统名分和更高层的镇邪追魔。

三者表面分工，实则互相渗透。

清平镇虽属王朝治下，但镇中有仙祠，凡涉香火、祭祀、修士斗法余痕、仙祠崩毁、邪祟异案等事，普通县衙与巡检司不敢擅断，通常要先报巡香司或庙观属吏处理。

所以开篇不是“官府不存在”，而是：

- 官面会来，但先到外围。
- 普通差役不敢碰仙祠核心。
- 巡香司和香火线更快进入核心现场。
- 真相往往先被能写卷宗的人决定怎么写。

## 15.2 权力结构总表

| 层级 | 代表机构 / 人物 | 明面职责 | 清平镇惨案中的实际作用 |
| --- | --- | --- | --- |
| 王朝地方行政 | 县衙、里正、甲首、保长 | 户籍、赋税、灾后统计、普通命案 | 登记死伤，报灾，安抚外围，不敢深入仙祠核心 |
| 王朝武力执行 | 巡检司、差役、乡勇 | 巡夜、抓盗、封路、维持治安 | 封镇口、拦乱民、守外围，等懂香火异案的人来 |
| 王朝特殊事务 | 巡香司、录事、巡香使 | 勘验仙祠、核供册、定性祭祀异案 | 真正进入核心废墟，收愿纸、香灰、供册、残符 |
| 地方庙观体系 | 庙祝、观主、香火执役、祝司首 | 祭典、供奉、香火账册、民间信仰接口 | 怕普通官面看见真相，倾向封口、改账、压案 |
| 宗门外围 | 外门执事、护境堂、斩魔司 | 镇邪、追魔、处理修士事务 | 更关心禁物、裂印、功绩，不优先关心凡人死伤 |
| 宗门高层 | 内门、功簿司、巡律堂、长老 | 法统、功绩、资源分配、大案定性 | 前期不直接到场，通过卷宗、命令和下层机构影响现场 |

## 15.3 王朝地方行政

县衙负责凡俗秩序。

包括：

- 户籍。
- 税粮。
- 田地。
- 灾后统计。
- 普通刑案。
- 赈济与安置。

但遇到以下情况，县衙通常不敢擅断：

- 仙祠崩毁。
- 香火断绝。
- 修士斗法。
- 魔修踪迹。
- 死状明显不合常理。
- 供册、愿纸、祭器涉及宗门或庙观名册。

在清平镇惨案中，县衙最合理的表现是：

- 会派人来。
- 会催死伤数字。
- 会要求封外围。
- 会等巡香司定性。
- 不会第一时间派县令深入塌毁仙祠。

写法重点：

> 县衙不是无能到不存在，而是被这类案子的规则压在外围。

## 15.4 巡检司与地方小吏

巡检司是凡俗层面能动武的执行力量。

职责：

- 封路。
- 巡夜。
- 抓盗。
- 控制流民骚乱。
- 配合县衙守住现场外围。

但巡检司对仙祠异案的态度是“围而不碰”。

他们可以站在镇口拉绳、贴封条、盘问幸存者，却不敢进仙祠核心翻供台、动神像、扫香灰。

里正、甲首、保长更适合承担在地事务：

- 认尸。
- 报灾。
- 统计谁家绝户。
- 指认镇中户籍。
- 协助差役问话。

他们话语权弱，容易被巡香司或庙观压着改口。

## 15.5 巡香司定位

巡香司是本书最重要的灰色中间层。

它不是单纯县衙机构，也不是纯宗门下属，而是：

> 王朝备案、庙观运转、宗门影响的香火异案机构。

在玄霄宗势力范围内，巡香司表面上有王朝特殊事务名分，实操上受地方庙观与玄霄宗香火脉牵制。

明面职责：

- 核验仙祠。
- 勘验香火异案。
- 管愿册、供册、香火账。
- 封场。
- 向县衙、庙观、宗门递报。

暗中作用：

- 清理不利于正统叙事的现场。
- 回收愿纸、香火灰、供册、残符。
- 改写案词。
- 压低死伤数字。
- 将复杂事件统一定性为“魔祸”“灾祟”“失祭反噬”等。

内部层级可先保持简单：

| 职位 | 功能 |
| --- | --- |
| 小吏 / 录事 | 抄册、跑腿、传话、记录死伤 |
| 查痕人 | 查灰、查愿纸、查供台残痕，罗灰属此类 |
| 巡香使 | 到场勘验、封场、判断香火异常，孟香使属此类 |
| 司首 / 祝司首 | 掌地方香火账、改案、压案、联通庙观与宗门 |

## 15.6 庙观与宗门的关系

地方庙观不是单纯拜神处。

它也是：

- 地方信仰接口。
- 香火归集点。
- 供奉名册存放处。
- 宗门治下标记。
- 王朝基层稳民心的工具。

庙观和王朝互相需要：

- 王朝需要庙观稳民心、解释灾异、安抚地方。
- 庙观需要王朝承认其合法性，方便收供、建祠、开祭。

庙观和宗门互相借力：

- 宗门不必直接治理每个小镇。
- 庙观借宗门牌子压地方。
- 宗门借庙观香火维持正统、名声和治下标记。

因此，清平镇出事后，宗门不必第一个到场。它可以让巡香司和庙观先收场，自己再拿走真正关心的禁物、功绩和案词结果。

## 15.7 清平镇惨案到场顺序

第一批：本地幸存者。

- 沈逐。
- 陈瘸子。
- 韩猎户。
- 周哑女。
- 少数濒死镇民。

他们最先看见废墟，也最先沾上真相。

第二批：地方最低层秩序。

- 里正。
- 巡检司差役。
- 报灾小吏。
- 乡勇。

他们会封外围、问死伤、催上报，但不敢碰仙祠核心。

第三批：巡香司 / 香火线处理者。

- 录事。
- 查痕人。
- 罗灰。
- 孟香使。

他们真正看供台、神像、香炉、旧庙、愿纸、香灰和供册。

第四批：庙观 / 宗门外围接口。

- 庙观执役。
- 外门执事。
- 护境堂或斩魔司留下的接头人。

他们确认昨夜修士大战到底动了什么，关心魔修、禁物、裂印、功绩和外泄风险。

第五批：更高层。

- 祝司首。
- 功簿司人物。
- 巡律堂调查者。
- 宗门更高层。

这批不宜前几章抢戏，只通过卷宗、命令、信物、调查令逐步浮出。

## 15.8 前十章使用口径

前十章不要让真正的王朝高层压进核心现场。

最合适的写法是：

- 第 1—4 章保持灾后封闭感。
- 第 5—8 章让地方小吏或巡检司外围短暂出现，说明王朝体系存在。
- 他们看到仙祠崩毁和修士残痕后退到外围。
- 罗灰、孟香使这类巡香司人员随后进入核心，形成“官面只能站外面，清痕人反而能进来”的压迫感。

可用现场细节：

- 差役贴封条，却不敢贴到神像裂缝旁。
- 里正拿着死伤簿，迟迟不敢写“仙祠塌毁”四个字。
- 巡检司的人站在镇口喊封镇，眼神避开供台。
- 巡香司录事拿出勘祠令，普通差役立刻让路。
- 孟香使先问愿纸在哪，而不是问还有多少活口。

## 15.9 可复用道具与文书

王朝与巡香体系要具体化，最好反复使用文书和道具。

常用道具：

- 报灾小牒。
- 死伤簿。
- 封场木牌。
- 巡检司封条。
- 勘祠令。
- 香火供册。
- 愿册残页。
- 定案案词。
- 上报朱批。
- 巡香司铜牌。

这些东西的作用不是堆名词，而是让“真相如何被改写”变成能看见的动作。

例如：

- 死了二百七十六人，死伤簿上只写一百九十三。
- 仙祠崩毁被改成“魔祸焚祠”。
- 供册缺页被说成火中遗失。
- 勘祠令一亮，县衙差役就不能再碰现场。

## 15.10 写作原则

- 官面要存在，但不要太快接管核心。
- 王朝基层要真实，不能写成全员空气。
- 普通差役可以怕，可以滑，可以按流程，但不必都坏。
- 巡香司的可怕在“专业”和“合法接口”，不是疯癫。
- 庙观的可怕在它既像信仰，又像账房和衙门。
- 宗门的可怕在它可以不亲手改账，却享受改账后的功绩。

一句话执行标准：

> 清平镇不是没人来管，而是来的人各管一层；越能碰核心的人，越不急着还死者真相。
