# 《诡仙录》宗门与修行势力

## 13.1 总原则

长篇不能只有玄霄宗一个宗门。

玄霄宗负责开篇的“庇护失职、争功、清痕、香火账”问题，但随着主角离开清平镇，修行世界必须逐层展开：

- 地方庇护宗门。
- 相邻竞争宗门。
- 中立商道宗门。
- 神道 / 香火类宗门。
- 魔道 / 异道势力。
- 旧仙脉残存势力。
- 上层仙盟 / 天榜 / 正统裁定机构。

每个宗门都不能只是换名字的敌人。它们要代表不同规则、资源、价值观和冲突方式。

宗门设计原则：

- 每个宗门绑定一种“秩序问题”。
- 每个宗门提供一种资源或修行入口。
- 每个宗门都要有内部灰度。
- 主角不正式投靠任何一宗，但会借用、伪装、交易、潜入、审判。

## 13.2 宗门层级

第一层：地方庇护宗门

- 管辖边荒镇村。
- 接受岁供、供灯、香火和地方归附。
- 最容易出现“庇护承诺失效”的问题。

第二层：区域竞争宗门

- 与玄霄宗争地盘、争功绩、争秘境、争香火边界。
- 能让正道内部不再铁板一块。

第三层：功能型宗门

- 商道、丹道、符道、剑道、律法、医修等。
- 提供资源、试炼、交易和身份伪装空间。

第四层：香火 / 神道宗门

- 更深地介入愿力、仙名、真名、庙观。
- 是第三卷以后“香火成网”的重要对手或舞台。

第五层：魔道 / 异道势力

- 不是统一魔教，而是多个旁门异道。
- 有的残忍，有的有规矩，有的只是被正道打成魔。

第六层：上层正统机构

- 类似仙盟、天律院、封名台、功德院。
- 决定什么叫正，什么叫邪，什么能受香火。
- 终局会和仙名秩序相连。

## 13.3 正道宗门：玄霄宗

定位：

- 开篇地方庇护宗门。
- 清平镇名义上的保护者。
- 第一至第三卷的重要矛盾源。

主问题：

- 除魔记功体系成熟。
- 护境承诺名存实亡。
- 巡香司负责收尾，把凡人死伤写进“魔祸损耗”。

代表机构：

- 护境堂。
- 斩魔司。
- 功簿司。
- 巡律堂。
- 巡香司。

代表人物：

- 许照骨。
- 顾清棠。
- 许观澜。
- 陆承言。
- 裴守拙。
- 商不言。

剧情功能：

- 第一卷：惨案源头和清痕体系。
- 第二卷：主角借壳入局，顾清棠查案。
- 第三卷：功簿、供册、庙观网络展开。

## 13.4 相邻正道宗门：青衡剑宗

定位：

- 玄霄宗相邻竞争宗门。
- 正统剑修大宗，重剑名、战功、门风。
- 不直接管清平镇，但会插手边境斩魔功劳。

主问题：

- 他们不靠香火吃人，但极重“剑名”和“战绩”。
- 在争夺斩魔功劳时，也会默认凡人损耗可以忽略。
- 宗门风气比玄霄宗更硬、更直，但同样会被功名推偏。

代表人物方向：

- 青衡年轻剑修：真有侠气，但看不起诡道。
- 青衡执剑长老：认剑理，不认凡人证词。
- 青衡弃徒：可作为主角中期临时对照，证明剑道正统也会逐人出门。

剧情功能：

- 第二 / 第三卷作为外部正道竞争压力。
- 可以让玄霄宗把清平镇惨案压下去的理由更复杂：怕旁宗抢功，也怕旁宗借题发难。
- 主角可借两宗争功互相牵制。

## 13.5 中立商道势力：万宝楼

定位：

- 跨城商会 / 坊市巨头。
- 不算宗门，但在修行界影响极大。
- 提供交易、拍卖、情报、身份伪装。

主问题：

- 他们不讲正邪，只讲价。
- 什么都能卖，包括功法、丹药、旧庙残物、香火器具、死人名册。
- 他们可以帮主角，也可以把主角卖出去。

代表人物方向：

- 楼中掌柜：笑面商人，知道很多事但从不白说。
- 情报女修：和顾清棠形成不同查案方式。
- 拍卖执事：让诡仙资源进入公开交易场。

剧情功能：

- 第二卷主角伪装入城的重要资源渠道。
- 第三卷提供府城和宗门间情报。
- 第四卷可出现旧庙残物拍卖，引出旧脉线索。

## 13.6 丹医宗门：回春谷

定位：

- 医修、丹修、救治体系。
- 表面最接近“护人”的正道势力。

主问题：

- 医修也受资源和宗门地位限制。
- 他们可能愿意救人，但救谁、救多少、用什么药，仍受功德、灵石、身份影响。
- 可以展开“善意也被资源分配卡住”的灰度。

代表人物方向：

- 回春谷医修：真想救人，却被要求先救宗门弟子。
- 丹房执事：以药价决定人命轻重。
- 叛谷医者：因私救凡人被逐，可和主角短暂同行。

剧情功能：

- 第二 / 第三卷补明线修为、疗伤、丹药资源。
- 让主角看到正道中有救人的人，但规则依旧偏向修士。
- 可作为顾清棠动摇的重要外部证据。

## 13.7 符阵宗门：天箓院

定位：

- 符箓、阵法、册录、巡境图谱相关宗门。
- 与香火、庙观、地方标记关系很深。

主问题：

- 他们用符阵管理地方，却容易把活人变成图谱上的点。
- 清平镇这种地方，在他们眼中可能只是“护境阵图上一处失火节点”。

代表人物方向：

- 天箓院符师：技术中立，只关心阵图是否完整。
- 巡图使：能看见清平镇护境记录被改过。
- 失籍符童：被符阵名册抹掉的人，适合连接诡仙“失名”主题。

剧情功能：

- 第三卷揭开巡境图录与香火供奉技术逻辑。
- 第四卷连接“名”和“阵”的高阶规则。
- 主角可从天箓院术法里借壳学习明线符法，再用诡仙旧术改路。

## 13.8 神道香火宗门：太微灵庙

定位：

- 大型香火神道势力。
- 管理庙观、灵牌、愿力、神名、祭典。
- 是“香火成网”中后期重要势力。

主问题：

- 他们最懂众生愿力，也最容易把愿力变成秩序资源。
- 表面慈悲，实则可能比剑宗、玄霄宗更会吞香火。
- 和仙名秩序关系最深。

代表人物方向：

- 灵庙祝官：笑容慈悲，话术最会安抚凡人。
- 愿簿使：负责记录众生愿望，也负责删改愿望。
- 无名牌位守者：可能知道被抹去之人的名字。

剧情功能：

- 第三卷后半 / 第五卷关键舞台。
- 展开“香火不是信仰，而是可被管理、转移、吞用的资源”。
- 是主角公开问仙的重要对手。

## 13.9 魔道势力：逆灯门

定位：

- 开篇异道势力。
- 正道称“逆灯魔门”，内部可称“残灯道”或“守灯一脉”。

主问题：

- 他们确实危险，有血法和禁术。
- 但有一支保留“不借凡人命”的规矩。
- 他们可能保存过诡仙旧物，但不等于诡仙遗脉。

代表人物：

- 裴烬：开篇被追杀者。
- 桑婆：知道裴烬真正带走什么。
- 乌行客：追索裂印者，不是主角盟友。

剧情功能：

- 第一卷制造正魔反差。
- 第二 / 第四卷提供夺印压力。
- 证明主角不能简单站到魔道一边。

## 13.10 异道旁门：白骨观

定位：

- 真正偏邪的旁门势力。
- 擅尸骨、阴煞、祭骨、役尸。

主问题：

- 他们就是会伤凡人的恶势力。
- 用来防止读者误解“魔道都有规矩”。
- 让主角面对真正该杀的邪道。

代表人物方向：

- 白骨观祭骨人。
- 收尸道人。
- 被香火司暗中利用的邪修。

剧情功能：

- 第二 / 第三卷可做局中局。
- 正道可能把所有魔祸都推给白骨观，但其中有真有假。
- 主角可借他们暴露正道栽赃与真实邪恶混杂的问题。

## 13.11 旧仙脉残存：无名祠

定位：

- 不是完整宗门。
- 是诡仙遗脉残留节点。
- 以旧庙、无名牌位、残碑、断坛、灯芯、失名者愿力存在。

主问题：

- 旧脉残缺，有误导，有代价。
- 并非所有旧脉遗留都是善的。
- 有些残留可能已经被香火秩序污染。

代表存在：

- 殷老祠。
- 无名牌位守者。
- 旧庙残念。
- 失名之人。

剧情功能：

- 第四卷核心。
- 主角从“借用裂印”转向“补全一条断脉”。
- 后期“重开无名祠”可从此处升华。

## 13.12 上层正统机构：天律盟

定位：

- 正道上层裁定机构。
- 管正邪名分、宗门功绩、香火归属、仙名敕封。
- 不是普通宗门，而是多宗共同承认的正统平台。

主问题：

- 他们决定谁是正、谁是邪、谁能受香火。
- 他们的文书和裁定看似公正，实则可能维护仙名秩序。
- 终局前会成为主角必须面对的制度高墙。

代表机构方向：

- 封名台：裁定仙名。
- 功德院：统计功绩。
- 天律司：追缉异端。
- 仙籍阁：记录宗门与修士名分。

剧情功能：

- 第五卷公开问仙的重要对手。
- 第六卷连接更高层仙名秩序。
- 主角最终不是打败某宗，而是挑战“谁有资格定义仙”。

## 13.13 宗门出场顺序

第一卷：

- 玄霄宗。
- 巡香司。
- 逆灯门。
- 无名祠 / 旧庙只露残痕。

第二卷：

- 玄霄宗县城外围。
- 万宝楼。
- 回春谷可通过丹药 / 医修线轻微登场。
- 青衡剑宗可埋竞争压力。

第三卷：

- 天箓院。
- 太微灵庙。
- 功簿司与府城庙观网络。
- 青衡剑宗正式参与争功或问责。

第四卷：

- 无名祠 / 旧仙脉残存正式展开。
- 逆灯门余部加压。
- 白骨观或其他邪道旁门制造“真邪道”对照。

第五卷：

- 天律盟。
- 太微灵庙高层。
- 多宗公开站队。

第六卷：

- 天律盟上层。
- 仙名秩序核心。
- 各宗门以不同立场回应“重定仙名”。

## 13.14 宗门使用原则

- 不要一次性把所有宗门讲出来。
- 每个宗门第一次出场，都要和一个具体事件、资源或人物绑定。
- 宗门之间要有竞争，不要都像一个总反派的分部。
- 主角可以借宗门法、借宗门名、借宗门矛盾，但不能被任何一宗收编。
- 正道宗门要有好人，魔道势力要有真恶，灰度来自规则和选择，不是简单反转。
