# 《诡仙录》分卷

> 当前采用六卷结构。四卷版的问题是跨度太大，“诡路寻仇”和“问仙于世”中间缺少足够的升级台阶；六卷版把地方香火网、诡仙旧脉、公开问仙分别拆出来，更适合长篇网文连载节奏。

## 第一卷：仙祸临门

核心目标：信仰崩塌，主角得印入局。

关键词：灾后开篇、善后得印、被查求活、第一次引错、旧庙局杀。

主要内容：

- 主角从仙祠废墟死人堆里醒来。
- 正道争功、魔修压偏杀势的反差落地。
- 主角为全镇善后，在供台底得到诡仙裂印。
- 地方香火线派查痕人清理废墟。
- 主角从观痕、藏气、引错、借势中学会求活。
- 卷末完成第一次较完整的“旧庙局杀”，让孟香使或其核心手下吃大亏。

卷尾状态：

- 主角不再只是幸存者，而是真正入局者。
- 明线修为从感气推进到开脉附近。
- 暗线裂印“得印”稳定，逼近“识印”。
- 主角确认小镇惨案不是孤立事件，背后有更深香火线。

卷尾钩子：

- 孟香使背后的祝司首 / 地方庙观主事露出影子。
- 主角得到一份账册、令牌、残符或旧愿线索，指向县城庙观。

## 第二卷：诡路寻仇

核心目标：从残村走向县城，追查第一层仇人。

关键词：县城庙观、香火执事、识印初成、做局成长、第一层复仇。

主要内容：

- 主角离开小镇，进入县城、庙观、外门势力交错的地方。
- 祝司首或同级香火主事从影子变成明确目标。
- 主角发现“清痕、灭口、收愿、改账”是一套成熟流程。
- 正道外围人物登场：有人冷漠，有人动摇，有人只是被体系推着走。
- 主角开始主动设局，不再只是躲避追查。
- 卷末查出小镇惨案的第一层真凶，并完成一次真正意义上的复仇。

卷尾状态：

- 主角杀掉或扳倒第一层仇人。
- 裂印进入“识印”阶段，能更稳定使用观痕、藏气、引错、借势。
- 主角意识到：杀一个执行者解决不了问题。

卷尾钩子：

- 仇人临死前吐出“功簿 / 供册 / 除魔记功”的存在。
- 小镇惨案被证明与更大范围的香火供奉、宗门功绩有关。

## 第三卷：香火成网

核心目标：揭开地方香火体系如何吃人。

关键词：供奉链、功绩簿、庙观网络、凡人代价、掌印门槛。

主要内容：

- 主角追查县城之上的府城、庙观联盟或宗门外门。
- 香火不只是烧香，而是礼、贡、契、势共同形成的治理网络。
- 主角看见更多凡人村镇被“供奉、清痕、记功、封口”裹挟。
- 正道功绩体系正式展开：除魔记功、资源分配、晋升名额。
- 主角开始接触更强修士，单靠小聪明不够，必须补足明线修为。
- 女主或重要同伴可在此卷正式入局，代表另一种看待正道的视角。

卷尾状态：

- 主角从“为自己复仇”转向“追问这套体系为什么存在”。
- 明线修为推进到筑基 / 灵府门槛。
- 暗线裂印从“识印”逼近“掌印”。
- 主角掌握一次中型布局能力，能让一处庙观体系短暂失灵。

卷尾钩子：

- 主角发现诡仙遗脉并非魔道分支，而是曾经被仙名秩序抹掉的另一条仙路。
- 一处旧庙、残碑、断坛或无名牌位指向第四卷。

## 第四卷：旧脉重现

核心目标：找到诡仙遗脉，确认裂印真正来历。

关键词：旧庙、残碑、断坛、被抹去的历史、掌印、裂名雏形。

主要内容：

- 主角进入旧庙遗址、诡仙断脉禁地或被封存的古战场。
- 裂印不再只是保命外挂，而成为打开旧脉真相的钥匙。
- 诡仙遗脉的“观、藏、借、改”完整逻辑逐步展开。
- 主角发现旧仙道并非只有正道一种，所谓诡仙曾经也是仙路。
- 正道、魔道、诡道的灰度进一步扩大：不是谁天然正确，而是谁愿意承担众生因果。
- 高位存在开始注意到裂印，暴露风险明显增加。

卷尾状态：

- 主角进入“掌印”阶段，能稳定执局、藏因、锁痕。
- 开始触碰“名”的概念，为裂名做铺垫。
- 主角不再只问“谁害了我”，而是问“谁有资格被称为仙”。

卷尾钩子：

- 主角得到一段被抹去的诡仙旧史。
- 旧史指向仙名秩序的核心问题：众生供奉可能正在喂养某种更高层秩序。

## 第五卷：问仙于世

核心目标：把隐藏的真相推到世人面前。

关键词：公开问仙、香火崩裂、仙名失格、裂名、照骨。

主要内容：

- 主角不再只在暗处杀人，而开始让事实显形。
- 他追查大宗门、仙城、法会、祭天大典等更高层场景。
- 香火线与正道功绩线合流，形成真正的大局。
- 主角通过旧术让“仙名”和“事实”当众冲突。
- 这一卷的爽点不是单杀强敌，而是让一地人不再盲信一个仙名。
- 正道人物内部开始分裂：有人维护秩序，有人醒悟，有人反咬主角为邪。

卷尾状态：

- 主角进入“裂名”阶段，开始扰动身份、真名、香火锚点。
- 明线修为进入金丹 / 元婴附近，具备硬实力。
- 主角成为旧秩序公开通缉的异数。

卷尾钩子：

- 高层仙名秩序第一次正面回应主角。
- 主角看见“仙”背后的真正锚点，意识到最后要面对的不是一个宗门，而是一套天上秩序。

## 第六卷：重定仙名

核心目标：立新规则，完成终局。

关键词：照骨、倒果、执劫、仙名秩序、重定“仙”。

主要内容：

- 主角面对真正掌控仙名秩序的高位存在。
- 诡仙遗脉断绝真相完整揭开。
- 众生供奉、愿力、真名、功绩、正统之间的最终结构浮出水面。
- 主角不只是复仇，也不只是揭露，而是要让旧秩序失去继续受供的资格。
- 终局战不应只是战力对轰，而是因果、香火、仙名、众生愿力层面的总局。
- 主角以“执劫者”身份承接大因果，重定仙名。

结局方向：

- 不一定灭尽诸仙，而是重定“仙”的含义。
- 仙若受人香火，便必须看见人间。
- 主角成为新路的开路者，而不是单纯登顶强者。

终局回收：

- 老庙祝的朴素话语回响成主角立道之言。
- 第一卷的供灯、愿纸、神像、旧庙意象在终局以更高层方式回收。
- “争的人找不到，埋人的人得到”升华为“争名者失其名，担劫者重定名”。
