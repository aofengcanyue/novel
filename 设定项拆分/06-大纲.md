# 《诡仙录》大纲

## 6.1 全书六卷大纲

第一卷：仙祸临门

- 灾后开篇。
- 主角从死人堆爬出。
- 善后得裂印。
- 被查、躲藏、第一次引错、旧庙局杀。
- 情绪关键词：痛、惧、活下去。

第二卷：诡路寻仇

- 从残村走向县城，追查第一层仇人。
- 祝司首或同级香火主事从影子变成明确目标。
- 主角学会更稳定地观痕、藏气、引错、借势。
- 顾清棠、许观澜正式进入主线，正道内部裂缝开始显形。
- 陆承言从影子变成具体人，负责“接手、统合、压事”。
- 卷末完成第一层复仇。
- 情绪关键词：恨、忍、反击。

第三卷：香火成网

- 追查供奉链、功绩簿、庙观网络。
- 揭开地方香火体系如何吃人。
- 正道功绩体系、凡人代价、宗门外围秩序正式展开。
- 主角从复仇进入追问。
- 情绪关键词：疑、查、破网。

第四卷：旧脉重现

- 主角进入旧庙、残碑、断坛等诡仙遗脉场景。
- 确认诡仙不是魔道分支，而是被抹掉的另一条仙路。
- 裂印从保命外挂变成主线钥匙。
- 主角进入掌印，开始触碰“名”的层面。
- 情绪关键词：追问、见史、执局。

第五卷：问仙于世

- 主角把隐藏真相推到世人面前。
- 香火线与正道功绩线合流，仙名和事实公开冲突。
- 主角开始让一地人不再盲信一个仙名。
- 正道内部出现更明显分裂。
- 情绪关键词：公开、质问、裂名。

第六卷：重定仙名

- 揭开诡仙遗脉断绝原因。
- 主角掌握裂名、照骨、倒果等高阶诡术。
- 与仙名秩序正面碰撞。
- 情绪关键词：立道、执劫、重定。

## 6.2 第一卷五段式

第一段：出事

- 小镇供奉仙门。
- 主角送供归来见废墟。
- 老庙祝、阿禾等情感钉子死亡。
- 主角信仰崩塌。

第二段：得印

- 主角为全镇善后。
- 收尸、掩埋、清理仙祠。
- 在供台底发现裂印。
- 裂印认主，主角开始观到残痕。

第三段：被查

- 地方最低层秩序先到外围：里正、巡检司差役或报灾小吏只负责封镇、认尸、上报。
- 地方香火线随后进入核心，派人清理痕迹。
- 孟香使或其手下登场。
- 主角用藏气、残坛土、香火灰躲过第一次追查。

第四段：做局

- 主角开始主动查线。
- 旧愿纸、阴骨钉、香火灰串联成小局。
- 主角不正面杀，而是让敌人误判、走错、暴露。

第五段：收局

- 旧庙局杀。
- 孟香使吃亏或死亡。
- 主角获得更明确线索。
- 卷尾引向更高层祝司首 / 地方庙观主事。

## 6.3 第一卷章节节奏

1—10章：灾后开篇 + 得印 + 第一次藏气 + 旧庙守人。

- 第1章直接从主角在死人堆里醒来开始。
- 前五章完成惨案、正魔反差、裂印入身、裂印显影和第一次观痕。
- 第6章让里正、巡检司差役或报灾小吏短暂出现在外围，说明王朝体系存在但不敢碰仙祠核心。
- 第7章让罗灰这个清痕人进入核心现场，形成第一层追查压力。
- 第8章完成第一次藏气求活，但罗灰必须留下“有人藏过”的判断。
- 第9—10章把阿禾旧愿纸和旧庙守人接上，为第一次引错铺线。
- 清晨开祠、送供上山、镇民许愿等内容改成碎片回忆穿插。

11—20章：孟香使定案 + 第一次引错 + 反噬查线 + 小局雏形。

- 第11章孟香使正式露面，压力从“清场流程”升级为“专业收尾人”。
- 第12章完成第一次“引错”，让主角在弱势中制造小爽点。
- 引错带来代价，孟香使开始真正注意主角。
- 主角围绕阿禾旧愿纸、旧庙、韩猎户、陈瘸子继续查线。
- 观痕、藏气、引错、借势开始串起来。
- 第二十章让主角确认“这场惨案连着更深香火线”，正式入局。

21—35章：查线 + 小局试手。

- 主角追查旧庙、愿纸、香火灰。
- 韩猎户、陈瘸子、殷老祠提供不同线索。
- 主角第一次主动准备完整小局。

36—42章附近：

- 陈瘸子完成“确认尸体不对、教主角人间看尸经验、替主角揽清痕”三项功能。
- 孟香使转入灭口流程后，陈瘸子成为第一批清场目标。
- 陈瘸子死后以白事簿残页继续留证，清平镇逐步变成“账上死镇”。

第二卷人物入场口径：

- 顾清棠先查：她在县城庙观偏殿 / 档房中核对死者名单、供奉账册和宗门记录。
- 许观澜先判：他真信除魔卫道，容易凭气息和教导先把主角归为邪路。
- 陆承言后压：他表面稳局，实际把不该消失的死人合法地消失掉。
- 裴守拙晚出：先通过批语、拒签记录或传闻埋影，不急着下场。

36—50章：旧庙局杀 + 卷尾反转。

- 主角利用旧庙地势、残坛土、旧愿纸、香火灰做局。
- 孟香使或其核心手下入局。
- 敌人不是因为蠢而入局，而是死在自己的贪、功绩压力和清痕流程惯性里。
- 结尾露出更高层黑线。

第一卷证据链单独维护在 [16-第一卷证据链.md](16-第一卷证据链.md)。大纲写章节功能时，应保证证据从异常感、阿禾旧愿纸、现场二次清理、孟香使定案，逐步升级到供册 / 名册 / 外地线索。

第一卷反派链条单独维护在 [17-第一卷反派链条.md](17-第一卷反派链条.md)。罗灰负责查痕流程，孟香使负责收尾定案，祝司首负责卷尾阴影，许照骨 / 斩魔司负责惨案源头压力。

第一卷所有小爽点必须遵循“不降智反派”原则：罗灰要专业，孟香使要老辣，祝司首要会远程控局。沈逐的胜利只能来自信息差、代价、地势、旧怨、证据缝隙和敌人的正确判断惯性，不能来自敌人犯低级错误。
