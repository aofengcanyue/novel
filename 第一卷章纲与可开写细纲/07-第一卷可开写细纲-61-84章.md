# 《诡仙录》第一卷可开写细纲：61—84章

> 单章目标约 2500 字。第61—68章完成旧庙局杀后半段，第69—84章处理余波、证据、离镇和第二卷钩子。

本段目标：

- 完成第一卷第一次完整诡道局杀：先乱、再错、最后收命。
- 让孟香使败在“必须收证、必须定案、必须压供册缺页”的职责惯性里，而不是败在降智。
- 让阿禾旧愿纸彻底反转为供册残页，完成情绪遗物到硬证据的转化。
- 卷尾拿到三类关键物：一个纸、一个名、一个外地祠场线索。
- 处理清平镇幸存者余波，回收陈瘸子遗页，让周哑女、韩猎户各自完成一段选择。
- 插入第71章和第83章顾清棠 / 叶知微 / 许观澜远景，给第二卷县城调查线接钩。
- 让沈逐离开清平镇，从幸存者变成带证据入局的人。

本段反派与卷尾口径：

- 孟香使在第61—67章仍要持续破局、压残愿、收证据，不能一进局就崩。
- 沈逐最终赢的是流程：孟香使的职责要求他必须收回供册残页，正是这点让他不能彻底退开。
- 罗灰第72章不必确认死亡，可保留“失去查痕能力 / 逃走余痕 / 灰匣证据”作为后续压力。
- 第一卷不能以“敌人死光了”收尾，而要以“流程还在、案词已定、新清场人将至”收尾。
- 顾清棠只看案卷，不救沈逐；许观澜仍倾向魔祸余孽，叶知微负责发现数字和页码错位。

## 第61章：孟香使的底牌

本章功能：

- 孟香使用巡香司法器或香火定案术压制旧庙残愿。
- 沈逐正面不敌，旧庙局险些被破。
- 阿禾旧愿纸残角成为唯一没有被压住的东西。

开场建议：

> 罗灰断线后，孟香使反而静了下来。

场景推进：

1. 孟香使确认罗灰无法继续报方位后，没有惊慌，立刻改用随身法器或香火定案术压场。
2. 他把旧庙浮起的残愿重新压回“魔祸残秽”的案词里，等于用制度口径压死人声音。
3. 沈逐借门槛灰、阴骨钉和旧灯芯维持旧庙局，却发现正面承压完全不如孟香使。
4. 孟香使一边压旧庙，一边锁定沈逐胸口残角，判断那才是旧愿和供册之间的关键。
5. 大部分残愿被压住，唯独阿禾留下的旧愿纸残角没有被完全压下。
6. 沈逐意识到，阿禾那一角不是普通愿纸，它能顶住孟香使的案词。

关键物件：

- 巡香司法器。
- 香火定案术。
- 阿禾旧愿纸残角。
- 旧庙残愿。

章末钩子：

> 满庙死灰都被压低。  
> 只有阿禾那一角纸，还在发冷。

## 第62章：阿禾留下的东西

本章功能：

- 阿禾旧愿纸显出真正残字，确认它是供册缺页一角。
- 孟香使急于收走，暴露最大破绽。
- 沈逐明白孟香使怕的不是死人，是账。

开场建议：

> 那张纸上的愿字，终于褪了。

场景推进：

1. 香火定案术压下普通愿意后，阿禾残角表层愿字被烧淡，露出下面更硬的供册格线。
2. 沈逐看见残角边缘的页印、供额残字和半个外地祠场名。
3. 孟香使立刻改变目标，不再优先杀沈逐，而是先要收走那角纸。
4. 他的反应太快，反而让沈逐确认：这纸牵涉的不是阿禾个人，而是供册和账。
5. 阿禾残愿短暂浮现，不说完整真相，只让沈逐看见她死前把纸攥进手心的动作。
6. 沈逐把残角贴近旧庙灰线，旧庙残愿开始顺着“账”而不是“怨”醒来。

关键物件：

- 供册格线。
- 页印残字。
- 外地祠场半名。
- 阿禾残愿。

章末钩子：

> 他怕的不是死人。  
> 是账。

## 第63章：账比命重

本章功能：

- 沈逐用供册残角引孟香使错判目标。
- 孟香使为了夺纸踏进旧庙失衡点。
- 旧庙灯芯重新亮起，局心合拢。

开场建议：

> 孟香使第一次没有先看沈逐。  
> 他看的是纸。

场景推进：

1. 沈逐故意让供册残角露出一息，像是自己将用它做局眼。
2. 孟香使判断这东西一旦落入旧庙残愿，清平镇案词会被撕开，所以必须抢在局成前收回。
3. 他避开前面两处假痕，选择一条看似最稳的收证路径。
4. 这条路不是沈逐凭空造的，而是孟香使职责决定的最短路线：先收纸，再压人，再封庙。
5. 他为了夺纸踏进神龛背后旧灰与偏殿退路之间的失衡点。
6. 旧庙断灯芯重新亮了一瞬，门槛灰、阴骨钉和供册残角终于连成一条线。

关键物件：

- 供册残角。
- 旧庙失衡点。
- 断灯芯。
- 神龛背后旧灰。

章末钩子：

> 灯芯亮了。  
> 不是照路，是照账。

## 第64章：第一重错

本章功能：

- 揭示第一重错：孟香使以为旧愿纸是局眼，实际上它只是引线。
- 沈逐借的不是纸本身，而是孟香使必须收证的职责惯性。
- 孟香使发现自己不是被术骗，而是被流程拖住。

开场建议：

> 孟香使伸手那一刻，沈逐松开了纸。

场景推进：

1. 孟香使用法器锁住供册残角，以为只要断纸，旧庙局就会散。
2. 残角被锁住后，旧庙残愿没有散，反而转向孟香使掌心的分装匣和批痕。
3. 沈逐这才明白，阿禾残角只是引线，真正被旧庙认住的是“谁在收账、谁在改账”。
4. 孟香使反应极快，立刻想抛开残角和分装匣，切断自己与供册缺页的关系。
5. 但他的职责惯性让他不能真的丢证，因为祝司首上令要求供册缺页不得外流。
6. 孟香使意识到，自己不是被沈逐的术骗住，而是被自己的收尾流程拖住。

关键物件：

- 被锁住的供册残角。
- 孟香使分装匣。
- 祝司首上令。
- 收证职责。

章末钩子：

> 孟香使忽然明白。  
> 骗他的不是术，是他自己办案的手。

## 第65章：第二重错

本章功能：

- 第二重错启动：孟香使误判退路。
- 他破开一层，却踩进另一层。
- 阴骨钉开始收紧。

开场建议：

> 孟香使退得很快，也退得很对。  
> 可沈逐等的，就是他这一步对。

场景推进：

1. 孟香使判断局心已偏向神龛，立刻选择退往偏殿断灯处，以法器重新压场。
2. 这一步若在正常庙观里完全正确，因为偏殿通常是香火残势最弱处。
3. 但旧庙的偏殿断灯早被殷老祠点明，是三处不能踏之一。
4. 孟香使破开沈逐故意留在神龛前的浅层旧灰，却踩进断灯与阴骨钉之间的第二层错。
5. 阴骨钉不是锁他肉身，而是锁住他“退一步再定案”的节奏。
6. 他终于被迫慢下来，旧庙残愿有了真正咬住他的时间。

关键物件：

- 偏殿断灯。
- 阴骨钉。
- 神龛前浅层旧灰。
- 孟香使退路判断。

章末钩子：

> 阴骨钉没有钉住他的脚。  
> 钉住的是他退开的那半步。

## 第66章：收命之前

本章功能：

- 沈逐重伤，旧庙局也将崩。
- 他用自己的血补住最后一处引线。
- 裂印在命里多出一笔，逼近得印稳定。

开场建议：

> 局成之前，先崩的是沈逐。

场景推进：

1. 孟香使被拖慢后，旧庙残愿开始咬住他，但沈逐自身也承受三重反噬。
2. 供册残角太脆，阴骨钉旧怨太乱，旧庙灯芯亮得不稳，整个局随时会散。
3. 孟香使看出沈逐撑不住，试图以香火定案术反压沈逐，让局债归到主角身上。
4. 沈逐必须用自己的血补住最后一处引线，等于让旧庙承认他也是这场旧账的承接人。
5. 裂印在命里多出一笔，疼痛不是升级爽感，而像把一段死人账刻进魂里。
6. 他稳住最后半息，等孟香使真正落入收命位置。

关键物件：

- 沈逐的血。
- 裂印新笔。
- 旧庙局债。
- 香火定案术反压。

章末钩子：

> 裂印多出一笔。  
> 像有人在他命上，写了一个债字。

## 第67章：收命

本章功能：

- 完成第一卷第一次完整诡道局杀。
- 展示诡道流程：观、藏、借、改、收。
- 孟香使临死前喊出祝司首。

开场建议：

> 沈逐没有出刀。  
> 他只是把最后一口气，藏进了灰里。

场景推进：

1. 观：沈逐看清孟香使身上最重的线不是修为，而是供册、案词、祝司首上令。
2. 藏：他藏住真正杀意，只让孟香使以为自己还在护残角。
3. 借：他借旧庙残愿、阴骨钉旧怨、阿禾残角和孟香使收证职责。
4. 改：他不改结果，只让孟香使每一步正确选择都滑向旧庙失衡点。
5. 收：阴骨钉、旧庙灯灰、裂印旧术同时合拢，孟香使被自己的案词和旧账反噬。
6. 孟香使临死前没有求饶，而是下意识喊出“祝司首”，像在怕更高处的追责。
7. 沈逐没有胜利快感，只听见旧庙里无数被压掉的名字短暂响了一瞬。

关键物件：

- 阴骨钉。
- 旧庙灯灰。
- 阿禾供册残角。
- 祝司首之名。

章末钩子：

> 孟香使临死前喊的不是救命。  
> 是祝司首。

## 第68章：反噬

本章功能：

- 局杀后反噬爆发，兑现诡道代价。
- 沈逐险些把自己也留在旧庙。
- 裂印多出一笔，得印阶段稳定。
- 让殷老祠只“扶半步”，不抹除局杀代价。

开场建议：

> 孟香使倒下后，旧庙没有安静。

场景推进：

1. 孟香使死后，旧庙残愿短暂失控，所有被借来的怨、灰、愿都开始向沈逐讨债。
2. 沈逐吐血、失眠感提前爆发，眼前清平镇、外地祠场、旧庙三案重叠。
3. 他险些分不清自己是沈逐，还是某个被改名册的人。
4. 阿禾残角贴在胸口，替他稳住一线，不是救命神物，而像有人提醒他还要把账带出去。
5. 殷老祠没有现身救场，只在局心外留下半截断香和一盏将灭的残灯，让沈逐看见离开的方向。
6. 沈逐借这点灯灰勉强离开局心，代价是阴骨钉碎裂、旧庙灰耗尽大半，旧庙债仍落在他命里。
7. 裂印在命里多出一笔，沈逐能更稳地观残和藏气，但也更容易被高位香火线察觉。

关键物件：

- 失控残愿。
- 阿禾残角。
- 碎裂阴骨钉。
- 裂印新笔。
- 半截断香。
- 将灭残灯。

章末钩子：

> 他没有变强多少。  
> 只是终于能背着这笔账，站起来。

## 第69章：搜身

本章功能：

- 从孟香使身上搜出关键证据。
- 证据链从灰、纸、尸体升级到账和名。
- 外地祠名出现，卷尾黑线明确。

开场建议：

> 沈逐搜孟香使的身，比杀他更难。

场景推进：

1. 沈逐强忍反噬搜身，不贪法器，只找文书、分装匣、供册残页、批痕。
2. 孟香使身上有自毁封记，沈逐只能在封记烧尽前取出几件最重要的东西。
3. 他拿到勘祠令、半页供册残页、分装匣残灰和祝司首批痕。
4. 供册残页上出现一个不属于清平镇的祠名，字迹与阿禾残角能拼上半边。
5. 沈逐意识到，这不是战利品，而是会引来更多追杀的东西。
6. 旧庙外传来动静，说明外围人已经察觉孟香使出事。

关键物件：

- 勘祠令。
- 供册残页。
- 分装匣残灰。
- 祝司首批痕。

章末钩子：

> 残页上那个祠名，不在清平镇。

## 第70章：供册残页

本章功能：

- 旧愿纸与供册残页拼合，阿禾线完成硬证据转化。
- 祝司首批痕确认上级存在。
- 沈逐第一次真正拿到能带出清平镇的“账”。

开场建议：

> 阿禾留下的半角纸，终于找到了另一半。

场景推进：

1. 沈逐把阿禾残角与孟香使身上的供册残页对上，纸纹、页印、药味、裁边全部能合。
2. 拼合后出现清平镇供额、祭期、户名缺口和外地祠场半名。
3. 残页背面有祝司首批痕，说明供册缺页不是孟香使临时私藏，而是上级默许或命令。
4. 阿禾当初不是留下愿望，而是留下能证明死者名册和供册对不上的证据。
5. 沈逐想起阿禾清晨许愿的碎片，情绪回收，但不写成长哭，只写他把纸折得很慢。
6. 他决定这张纸必须活着出镇。

关键物件：

- 拼合供册残页。
- 清平镇供额。
- 祝司首批痕。
- 阿禾残角。

章末钩子：

> 阿禾没有把愿望留给仙。  
> 她把账留给了他。

## 第71章：外地祠名

本章功能：

- 外地祠场名明确指向县城辖下另一处仙祠。
- 清平镇不是孤例。
- 顾清棠远景短镜头呼应旧案格式。

开场建议：

> 那个祠名，沈逐没听过。

场景推进：

1. 沈逐辨认残页上的外地祠场名，只能看出它属于县城辖下某处仙祠或乡祠。
2. 香牌编号、罗灰灰匣旧灰、供册残页上的祠名终于形成三点对应。
3. 他判断孟香使或祝司首至少处理过两处类似案场。
4. 远景短镜头：县城档房，顾清棠翻到另一份旧案，看见相似祠名和相似案词格式。
5. 叶知微提醒那案也写作“魔祸焚祠”，死伤附页却同样缺失。
6. 顾清棠没有下结论，只把两个案卷的祠名和页码抄在同一张纸上。

关键物件：

- 外地祠名。
- 香牌编号。
- 另一份旧案。
- 顾清棠抄下的页码。

章末钩子：

> 那处祠场，也曾被写作“魔祸”。

## 第72章：罗灰的余痕

本章功能：

- 处理罗灰余波，不让敌人凭空消失。
- 留下灰匣证据和后续追杀压力。
- 说明巡香司还会有人来。

开场建议：

> 罗灰没有死在灰门里。

场景推进：

1. 沈逐在灰门附近找到罗灰逃走或被拖走的余痕，痕迹断续，说明他失去了稳定查痕能力。
2. 罗灰的灰匣碎裂，里面洒出第三处旧灰，气味既不属于清平镇，也不属于第71章外地祠名。
3. 沈逐没有追杀罗灰，一是没余力，二是外围巡香司和官差已经逼近。
4. 罗灰若活着，会带回“旧印在沈逐身上”的模糊判断；若撑不住，也会留下足够让巡香司继续追查的灰证。
5. 沈逐确认，孟香使死了不等于事情结束，流程会自动补人。
6. 他收起灰匣碎片，把第三处旧灰包好。

关键物件：

- 罗灰余痕。
- 碎裂灰匣。
- 第三处旧灰。
- 旧印模糊判断。

章末钩子：

> 清平镇后面，还有第三处灰。

## 第73章：陈瘸子的遗页

本章功能：

- 回收陈瘸子死前藏下的白事簿残页。
- 让陈瘸子的死从“替主角挡刀”变成“用命留下证词”。
- 明确清平镇正在变成账上死镇。

开场建议：

> 陈瘸子没有来。
> 来的是一盏灭了的白事灯。
